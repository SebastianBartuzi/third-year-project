README

ADDING SIMULATOR TO UNITY:
In order to add the project to Unity, go to Unity Hub and select Open -> Add project from disk and select the folder with the project.

RUNNING EXE:
If you want to run executable file of the simulator, go to the folder and open the Simulator file.

ATTENTION: The project will run correctly only on machines with Windows OS and with SWI-Prolog installed (otherwise, recommendations will not be displayed on the screen). If you do not have SWI-Prolog installed, go to https://www.swi-prolog.org/download/stable and follow the instructions.
